Test Instructions for Catharsis Feature

Steps to Test:

  1. Create a new journal entry
    • Click "New Entry" button
    • Add a title like "Stressful Day at Work"
  2. Write some test content with negative emotions:
    "Today was absolutely terrible. My boss was being completely unreasonable about the deadline. I'm feeling so overwhelmed and anxious about everything. The traffic was awful this morning and made me really angry. I just feel sad that things keep going wrong."
  3. Test the catharsis feature:
    • Select the word "terrible" - popover should appear above it
    • Choose "Angry" stress level
    • Select "overwhelmed and anxious" - choose "Anxious"
    • Select "really angry" - choose "Angry"
    • Select "feel sad" - choose "Sad"
  4. Open Catharsis Window:
    • Click the heart button in toolbar (should show count like "Catharsis (4)")
    • Window should open showing all tagged stress items
  5. Test therapeutic disposal:
    • Try dragging an item to the trash icon
    • Try clicking "Shred" button on an item
    • Should hear appropriate Mac sounds
    • Items should disappear from the list
    • Count in toolbar should update
  6. Save the entry
    • Click "Save Entry"
    • Remaining catharsis items should be saved with the entry

Expected Behavior:

  • ✓ No regex errors when selecting text
  • ✓ Stress level popover appears above selected text
  • ✓ Text gets highlighted with appropriate stress colors
  • ✓ Catharsis window shows all tagged items
  • ✓ Drag and drop to trash works
  • ✓ Shredding with sound effects works
  • ✓ Item count updates correctly
  • ✓ Entry saves with catharsis data
